# re-magic-layout
> **re-magic-layout** 是一款基于React框架实现的一款多功能画布，主要使用场景为编排置入其中的组件，致力于成为前端可视化搭建领域最易使用和最易扩展的React画布。

## 已完成功能 ✨
1. [x] 将传入的子元素，包裹成可拖拽的单元
2. [x] 实现元素的自由拖拽，但是需要有边界（因为画布是有限的，以后再考虑无限画布的方案）
3. [x] 元素的拖拉调整宽高功能
4. [x] 元素拖拽时的辅助线功能
   - [x] 水平辅助线
   - [x] 垂直辅助线
5. [x] 子元素默认位置控制
